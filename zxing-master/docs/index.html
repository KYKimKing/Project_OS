About ZXing

Parent Maven project for ZXing modules

Project Modules

This project has declared the following modules:

Name Description
ZXing Core Core barcode encoding/decoding library
ZXing Java SE extensions Java SE-specific extensions to core ZXing library
ZXing Android Core Code common to Android applications
ZXing Android Integration lib Integration helper classes for Android applications wanting to create/scan barcodes
ZXing Appspot-based encoder GWT-based encoder app hosted at zxing.appspot.com
ZXing zxing.org web app The web application behind zxing.org
